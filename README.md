# 🍽️ Spice Delight - Restaurant Website

A modern and responsive restaurant website built using React.js. This project showcases a restaurant's menu, services, featured dishes, and contact information with an attractive and user-friendly interface.

## 🚀 Features

- 🏠 Modern Home Page
- 🍔 Featured Food Items & Menu Section
- 📖 About Us Section
- 📞 Contact Form
- 📱 Fully Responsive Design
- 🎨 Clean and Attractive UI
- ⚡ Fast Performance

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router DOM
- React Icons

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/restaurant-portfolio.git
```

2. Navigate to project folder

```bash
cd restaurant-portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

The application will run at:

```bash
http://localhost:3000
```
